# Frequently Asked Questions

## What is ARDaC?

ARDaC is a data commons that helps users discover, access, and analyze project data through a governed platform.

## Who can use ARDaC?

Access depends on project policy. Some resources may be public, while protected datasets require approval.

## How do I search for data?

Use the Data Portal search page to find datasets, files, and metadata. Start with broad keywords and narrow results with filters.

## How do I request access to controlled data?

Open the dataset page and follow the access request instructions. You may need to provide project details or complete a data use agreement.

## Can I download files?

Yes, if the dataset allows downloads and your account has the required permissions.

## Can I use the API?

Yes. API access usually requires authentication and may require dataset-specific authorization.

## Where should I analyze protected data?

Use only approved environments that satisfy the dataset's access and security requirements.

## Who should I contact for help?

Use the support contact, help desk, or project channel listed by the ARDaC team. For technical deployment questions, refer to the Operator Guide. For customization and integration questions, refer to the Developer Guide.
