# Getting Started

This page is a starting point for researchers who are new to ARDaC.

## 1. Review the Project Context

Start with the public project sites:

- [ARDaC](https://ardac.org/) for data commons background, news, team information, and publications.
- [AlcHepNet](https://alchepnet.org/) for information about the Alcohol-associated Hepatitis Network.

## 2. Open the Portal

Go to [portal.ardac.org](https://portal.ardac.org/) to explore the ARDaC Portal.

Depending on the action you are taking, some portal features may require sign-in or approved access. Access requirements should be confirmed with the ARDaC team before relying on a specific workflow.

## 3. Find Available Resources

Use the portal's discovery and data exploration views to review available studies and data resources. Start broad, then narrow by study attributes, metadata, or available data types.

## 4. Get Help

For questions about portal access, available resources, or documentation gaps, contact [ardac@iu.edu](mailto:ardac@iu.edu).
