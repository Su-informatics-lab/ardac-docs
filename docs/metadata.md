# Metadata

Metadata helps researchers understand what data are available, how resources are organized, and whether a study or dataset is appropriate for a research question.

## Common Metadata Areas

ARDaC resources may describe information such as:

- study and cohort context
- participant or case-level attributes
- biospecimen collection details
- clinical conditions, laboratory tests, treatments, and outcomes
- assay or omics data categories
- data availability and access notes

## Data Dictionary Concepts

The portal may expose data dictionary or schema information that describes fields, expected values, and relationships between data entities. Use those details to understand terminology before comparing resources or planning an analysis.

## Documentation Gaps

If a metadata field, value, or portal term is unclear, contact [ardac@iu.edu](mailto:ardac@iu.edu). Include the page, study, field name, and a short description of what was unclear.
