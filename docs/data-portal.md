# Data Portal

The ARDaC Data Portal is the main web interface for discovering datasets, reviewing metadata, requesting access, and navigating available tools.

## Main areas

- **Home:** project overview, announcements, and entry points
- **Explore:** search and browse datasets and files
- **Dataset pages:** metadata, documentation, versions, and access details
- **Access requests:** status and instructions for protected datasets
- **User account:** profile, permissions, and linked services

## Portal content

Portal pages should be written for both new and experienced users. Use clear descriptions, consistent dataset names, and links to related documentation.

## Troubleshooting

If portal content appears missing or permissions look incorrect, confirm that you are signed in with the expected account and that your access request has been approved.
