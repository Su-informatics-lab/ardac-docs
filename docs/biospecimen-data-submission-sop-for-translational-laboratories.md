# Biospecimen Data Submission SOP for Translational Laboratories

## Purpose

This standard operating procedure describes how translational laboratories should prepare, validate, and submit biospecimen data to ARDaC. The goal is to ensure that biospecimen records are complete, consistent, traceable, and ready for integration with clinical, molecular, imaging, and analysis data.

## Scope

This SOP applies to laboratories submitting biospecimen metadata, collection details, processing information, storage information, shipment records, assay linkage identifiers, and related quality-control fields for ARDaC-supported studies or programs.

## Roles and Responsibilities

Submitting laboratories are responsible for preparing accurate data, using approved templates, resolving validation errors, and responding to ARDaC data-curation questions. Study data managers or project leads are responsible for confirming study-specific requirements. The ARDaC data team is responsible for reviewing submissions, running quality checks, and loading approved records.

## Required Submission Materials

Each submission should include the completed biospecimen data template, any required data dictionary or code-list references, study and protocol identifiers, submitter contact information, and supporting notes for non-standard fields or known exceptions.

## Data Preparation

Before submission, confirm that specimen identifiers are unique, subject or participant identifiers match the approved study records, collection dates follow the accepted format, specimen type and anatomic site values use controlled terminology, and processing or storage details are complete where required.

## Submission Steps

Download the current ARDaC biospecimen submission template. Complete all required fields and optional fields relevant to the study. Run the available validation checks before submitting. Submit the file through the approved ARDaC intake channel or data-submission portal. Include a brief summary of the submission scope, such as study name, batch number, specimen count, and any known issues.

## Validation and Review

ARDaC will review the submitted file for required fields, identifier consistency, controlled vocabulary compliance, date formatting, duplicate records, and logical relationships between specimen, aliquot, shipment, and assay-linkage fields. Validation questions will be returned to the submitter for correction or clarification.

## Corrections and Resubmissions

When corrections are requested, update the source file rather than sending isolated edits. Clearly mark the submission version and provide a short change summary. Resubmitted files should preserve stable identifiers unless ARDaC specifically requests an identifier correction.

## Acceptance Criteria

A biospecimen submission is ready for loading when required fields are complete, validation errors are resolved, controlled terminology is accepted, identifiers are traceable, and the submitting laboratory or project lead confirms the final version.

## Support

For questions about templates, controlled terms, validation errors, or submission status, contact the ARDaC data-submission support team and include the study name, submission version, and laboratory contact.
