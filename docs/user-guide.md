# User Guide

This guide introduces the main ARDaC workflows for finding data, requesting access, downloading files, and using analysis tools.

## Before you begin

Make sure you have an ARDaC account and the correct permissions for the datasets you plan to use. Some datasets may be public, while others require approval before file access is granted.

## Common user workflows

1. Sign in to the ARDaC Data Portal.
2. Search or browse available datasets.
3. Review dataset metadata, documentation, and access requirements.
4. Request access when required.
5. Download files or use approved analysis environments.
6. Cite or acknowledge datasets according to project guidance.

## Good practices

Keep track of dataset versions, document your analysis steps, and avoid moving controlled data into unapproved locations.
