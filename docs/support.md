# Support

For questions about ARDaC documentation, portal access, available data resources, or metadata, contact:

[ardac@iu.edu](mailto:ardac@iu.edu)

## Helpful Details to Include

When reporting a question or issue, include:

- the portal page or documentation page you were using
- the study, dataset, or metadata field involved
- the action you were trying to complete
- any error message or unexpected result
- your institutional affiliation, if relevant to the request
