# ARDaC Community

The ARDaC community includes researchers, data contributors, platform operators, developers, and partner teams working together to improve access to high-quality data resources.

## Ways to participate

- Share feedback about the data portal and user workflows
- Contribute documentation, examples, and training materials
- Report issues or request new features
- Join project meetings, office hours, or working groups
- Publish reusable notebooks, scripts, and analysis patterns

## Community expectations

Participants are expected to communicate respectfully, protect sensitive data, follow project governance requirements, and document decisions that affect shared resources.

## Getting help

For questions about using ARDaC, start with the User Guide and Frequently Asked Questions. For deployment, operations, or extension work, use the Operator Guide and Developer Guide.
