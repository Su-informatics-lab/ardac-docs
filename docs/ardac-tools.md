# Tools

ARDaC tools help users move from data discovery to access, download, analysis, and reporting.

## User tools

- Data Portal for search and browsing
- File access tools for approved downloads
- Metadata views and dataset documentation
- Analysis notebooks or workspaces, when enabled
- Export tools for manifests, metadata, and citations

## Developer and operator tools

- APIs and client libraries
- Ingestion and validation utilities
- Monitoring dashboards and logs
- Deployment scripts and configuration templates
- Documentation and release tooling

## Choosing a tool

Use the Data Portal for interactive discovery. Use APIs or command-line tools for repeatable workflows. Use approved analysis environments for protected data.
