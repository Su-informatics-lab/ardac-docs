# Blog

The ARDaC blog shares updates, tutorials, release notes, community stories, and examples of responsible data use.

## Suggested blog categories

- Project announcements
- New dataset releases
- User tutorials
- Technical updates
- Community highlights
- Governance and policy updates
- Analysis examples

## Blog post template

Each post should include a title, publication date, author or team, summary, main content, related links, and contact or feedback information.

## Editorial guidance

Write in clear language, avoid exposing sensitive information, and link to official documentation when explaining platform workflows.
