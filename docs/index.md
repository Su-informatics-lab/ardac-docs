# ARDaC Documentation

The Alcohol Research Data Commons (ARDaC) is the central data hub and research nexus for AlcHepNet, the Alcohol-associated Hepatitis Network. AlcHepNet is a collaborative research consortium supported by the National Institute on Alcohol Abuse and Alcoholism (NIAAA).

Use this documentation to get started with the ARDaC Portal, find available data resources, understand study metadata, and learn where to get help.

## What ARDaC Provides

ARDaC supports discovery and research use of AlcHepNet data resources through a web portal at [portal.ardac.org](https://portal.ardac.org/). The portal is intended to help researchers explore available studies, review metadata, and understand the data resources connected to the commons.

The public ARDaC website includes additional project background, news, team information, publications, and acknowledgements. The AlcHepNet website provides broader context about the research network and its alcohol-associated hepatitis studies.
