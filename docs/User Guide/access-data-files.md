# Access Data Files

ARDaC supports secure access to data files according to project policies, dataset permissions, and user roles.

## Access levels

Datasets may be open, registered, controlled, or restricted. The exact access model depends on project governance and data sensitivity.

## Requesting access

When a dataset requires approval, follow the access request instructions shown in the Data Portal. You may need to provide your role, project purpose, institutional affiliation, data use agreement, or compliance documentation.

## Downloading files

After access is granted, approved users can download files through supported portal tools, command-line utilities, or APIs.

## Handling controlled data

Store controlled files only in approved environments. Do not redistribute data, publish sensitive records, or combine datasets in ways that violate the approved use terms.
