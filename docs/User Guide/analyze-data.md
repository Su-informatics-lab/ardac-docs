# Analyze Data

ARDaC is designed to support reproducible analysis after users discover and access approved datasets.

## Analysis options

Depending on the project configuration, ARDaC may support local downloads, secure workspaces, notebooks, workflow tools, or integrations with external compute environments.

## Recommended workflow

1. Identify the dataset and version used in the analysis.
2. Confirm that your access approval covers the intended use.
3. Prepare the data in an approved environment.
4. Record scripts, parameters, software versions, and outputs.
5. Save reusable notebooks or workflows when appropriate.

## Reproducibility

Use clear file references, maintain versioned code, and include enough metadata for another approved user to understand and repeat the analysis.
