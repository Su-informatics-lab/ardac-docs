# Search for Data

ARDaC provides search and filtering tools to help users discover datasets, files, projects, and metadata records.

## Search options

You can search by keyword, dataset name, project, contributor, data type, file format, or other metadata fields made available by the project.

## Filtering results

Use filters to narrow results by access level, data category, cohort, study, date, file type, or other ARDaC-specific metadata.

## Reviewing dataset records

Before requesting or downloading data, review the dataset description, data dictionary, provenance, usage restrictions, citation guidance, and version information.

## Tips

Use broad terms first, then refine with filters. When results are unexpected, check spelling, alternate names, acronyms, and related project terms.
